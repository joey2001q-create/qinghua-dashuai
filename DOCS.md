# 清华大帅AI - 项目功能文档

> 本文档随开发进度同步更新，记录所有功能点及其实现状态

---

## 📋 项目概述

- **项目名称**：清华大帅AI（qinghua-dashuai）
- **技术栈**：Next.js 16 + React 19 + TypeScript + Tailwind CSS 4 + Recharts
- **AI模型**：讯飞 coding plan（astron-code-latest）
- **API**：SSE 流式输出

## 📖 学习规划模块 (`/plan`)

### 1. ABC目标提分表 (`/plan/abc`)

| 功能 | 状态 | 说明 |
|------|------|------|
| 添加科目目标 | ✅ | 科目、满分、当前分、目标分、信心指数 |
| ABC目标计算 | ✅ | A=理想目标, B=预期可达, C=当前分数 |
| 信心指数滑块 | ✅ | 1-10级，实时调整 |
| AI分析 | ✅ | 流式输出，分析提分策略 |
| 导出Markdown | ✅ | 导出完整ABC目标表 |
| Markdown渲染 | ✅ | 表格、列表美化 |

**公式**：提分 = 差距 × (信心指数 / 10)，B目标 = 当前分数 + 提分

### 2. 学习计划 (`/plan/schedule`)

| 功能 | 状态 | 说明 |
|------|------|------|
| 年级选择 | ✅ | 初一、初二、初三 |
| 学科/考试类型 | ✅ | 下拉选择 |
| 分数信息 | ✅ | 总分、目标分、当前分 |
| 备考天数 | ✅ | 距离考试天数 |
| 每日学习时间 | ✅ | 1-4小时选择 |
| Stream流式输出 | ✅ | SSE实时渲染 |
| Markdown美化 | ✅ | remark-gfm支持表格 |
| 进度条 | ✅ | 当前→目标分数进度 |
| 柱状图 | ✅ | 分数对比图表 |
| 导出 | ✅ | Markdown文件导出 |
| 分数验证 | ✅ | 不能负数、不超过满分 |
| 自由滚动 | ✅ | 流式输出时可滚动 |
